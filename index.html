Producteev PHP Library by greatwitenorth

This is a modified version of code originally created by Angelo R. of http://xangelo.ca. It allows you to easily use the Producteev api in PHP. I included some bug fixes and hope to develop it further.

Producteev is a full featured taks management system that allows users to collaborate with one another. They have provided a full featured API that allows you to extend their platform to suit your needs. This library will help you to use the API in PHP.

Install

Simply put the folder into a web server and navigate to the example.php file.
